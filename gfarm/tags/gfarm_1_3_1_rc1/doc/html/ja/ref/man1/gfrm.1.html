gfrm

Name

gfrm -- Gfarm ファイル、ディレクトリ、ファイル複製を削除する。

Synopsis

gfrm [-frR] [-I ファイルセクション] [-h ファイルシステムノード] gfarm-URL...

DESCRIPTION

Gfarm-URL に指定されたファイルを削除します。 デフォルトではディレクトリの削除は行いませんが、 -r あるいは -R オプションが指定された場合、 指定されたディレクトリ以下のファイルおよびディレクトリが再帰的に消去されます。 なお、. あるいは .. が指定された場合はエラーになります。

-Iオプションを指定することにより、 ファイルの特定のファイルセクションを削除することができます。 ファイルセクションとは、実行形式のファイルの場合はアーキテクチャ名、 通常ファイルの場合はファイルインデックス番号となります。

-h オプションでファイルシステムノードを指定することにより、 指定されたノードに格納されているファイル複製を削除することができます。 ただし、全てのファイル複製を削除することはできません。

OPTIONS

-f

ファイル複製削除の場合に、 全てのファイル複製を削除することになってもファイル複製を削除します。 この時、ファイル自体が削除されることになります。 このオプションは -h オプションとともに指定された場合に有効です。

-r, -R

引数がディレクトリだった場合、そのディレクトリ以下を 全て、再帰的に削除します。

-h ファイルシステムノード

複製を削除するファイルシステムノード名を指定します。 複数のファイルシステムノードか ら同じファイル断片の複製を一度に削除するためには，このオ プションを複数回指定します．

-I ファイルセクション

削除するファイルセクションを指定します。 ファイルセクションとは、 実行形式のファイルの場合は、アーキテクチャ名(例 「sparc-sun-solaris8」)、 通常ファイルの場合は、ファイルインデックス番号のことです。

-?

引数オプションを表示します。
