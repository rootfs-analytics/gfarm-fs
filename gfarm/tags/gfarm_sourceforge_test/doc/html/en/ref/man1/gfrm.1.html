gfrm

Name

gfrm -- remove Gfarm files, directories, or file replicas

Synopsis

gfrm [-frR] [-I file-section] [-h filesystem-node] gfarm-URL...

DESCRIPTION

gfrm removes each specified file. It does not remove directories unless the -r or -R is specified. '.' or '..' cannot be removed.

When -I option is specified, gfrm removes the specified file section. File section is an architecture name for executable files, or is a fragment index for a regular file.

With the -h option, gfrm removes file replicas of each specified file or file section on the specified filesystem node(s). All file replicas of a file cannot be removed.

OPTIONS

-f

Even when all file repicas are specified by the -h option, force to delete all the file replica. In this case, the specified file will be deleted.

-r, -R

remove all entries under the directories specified by the command-line argument recursively.

-h filesystem-node

specifies a filesystem node on which file replicas will be removed. With the -I option, a file raplica of the specified file section will be removed. This option can be used more than once to remove file replicas on several filesystem nodes at once.

-I file-section

specifies the file section to be removed. File section is an architecture name such as sparc-sun-solaris8 for executable files, and an fragment index for regular files.

-?

displays a list of command options.
