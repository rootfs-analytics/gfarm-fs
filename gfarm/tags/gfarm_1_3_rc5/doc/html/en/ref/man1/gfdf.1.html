gfdf

Name

gfdf -- display gfarm filesystem disk free space

Synopsis

gfdf -H hostfile [-i]

gfdf -h hostname... [-i]

DESCRIPTION

gfdf displays disk usage and free space of specified gfarm filesystem node.

OPTIONS

-H hostfile

specifies a list of filesystem nodes. The hostfile consists of a hostname of a filesystem node in each line.

if ``-'' is specified, standard input is used to read the host list.

-h hostname

specifies a hostname of a filesystem node. This option can be used multple times.

-i

displays i-node information instead of disk space information.

-?

displays a list of command options.

SEE ALSO
